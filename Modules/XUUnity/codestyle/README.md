# XUUnity Code Style

## Purpose
This folder defines the default shared code style for XUUnity work.

## Rule
Any AI work using XUUnity should load the relevant files from this folder before implementation or review.

## Typical Load
- `codestyle/csharp.md`
- `codestyle/unity.md`
- `codestyle/native.md` only if native code is involved

## Override Rule
Project-specific coding constraints in `Assets/AIOutput/ProjectMemory/` override these shared rules when there is a conflict.
