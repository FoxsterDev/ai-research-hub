# XUUnity UI Tweens Skills

## Purpose
Use `skills/ui_tweens/` for Unity UI animation systems and tween-library-specific safety rules.

## Focus
- tween target lifetime
- animation ownership
- popup and screen close-order safety
- callback allocation discipline
- debugging null or destroyed tween targets

## Modules
- `primetween.md` for PrimeTween-specific implementation and review guidance
- future tween-library modules should live here when they are public-safe and reusable
