# XUUnity Core Skills

## Purpose
`skills/core/` defines the non-optional baseline for all XUUnity work.
These files should load for every task before task-specific skills.

## Baseline Posture
- Unity `6000+` is the default engine baseline.
- Mobile production constraints are the default target.
- Zero crash and zero ANR are hard requirements.
- No microfreezes on critical user flows.
- Performance, allocation, startup, and thermal discipline are mandatory.
