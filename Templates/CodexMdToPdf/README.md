# Codex Markdown to PDF Prompt Templates

Reusable prompt templates for the local Markdown-to-PDF workflow.

These templates are public-safe and intentionally generic:

- they assume the local `CodexMdToPdf` operation is already installed
- they do not mention project-private paths
- they are intended for trusted local Markdown input only

Templates in this folder:

- `render_local_markdown_to_pdf.md`
- `render_local_markdown_to_pdf_to_output.md`
