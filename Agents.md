# 📂 Repository Structure

This document provides a quick navigation guide to the `ai-research-hub` repository.

## 📁 Root Directory
*   **`README.md`**: The main entry point. Contains the purpose of the repo and links to the latest critical reports.
*   **`Agents.md`**: (This file) Overview of the repository structure.

## 📁 reports/
Contains all generated analysis reports, organized by platform and SDK.

*   **`index.md`**: A catalog of all reports with high-level recommendations and scores.
*   **`unity/`**: Reports specific to the Unity game engine.
    *   **`appsflyer/`**: Analysis of the AppsFlyer SDK for Unity.
        *   `AppsFlyer_UnityIAP4.13_Analysis_2026-01-12.md`: Detailed analysis of version 6.17.80.

## 📁 docs/
Documentation regarding the research methodology and standards.

*   **`methodology/`**:
    *   `scoring-logic.md`: Explains how the "Production Ready" score (0-15) is calculated.

## 📁 templates/
Reusable templates for generating consistent reports.

*   **`report_template.md`**: The standard Markdown structure for new SDK analysis reports.
